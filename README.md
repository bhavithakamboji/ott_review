# OTT Review Moderation & Parental Control System

AI-powered review classification system using NLP and Machine Learning.

## Technologies Used
- Python
- Flask
- NLTK
- TF-IDF
- Logistic Regression

## Features
- Toxic review detection
- Spoiler detection
- Spam filtering
- Age-based parental control

## How to Run
pip install flask pandas scikit-learn nltk
python app.py
